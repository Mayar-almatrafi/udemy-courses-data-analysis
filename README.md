# Udemy Courses Data Analysis (EDA)

Exploratory Data Analysis (EDA) on a Udemy courses dataset to understand trends in **course subjects, pricing, engagement, and popularity**.  
This project focuses on **cleaning**, **validating**, and **analyzing** the data, then communicating insights through simple visualizations.

## Project Overview
This repository contains a Jupyter notebook that walks through:
- Data cleaning (duplicates, formatting, validation)
- Descriptive statistics (central tendency)
- Correlation analysis
- Grouping & aggregation (by subject / level / paid vs free)
- Text analysis (basic word frequency + word cloud)
- Visualizations (histograms, scatter plots, summary charts)

  ## Files
- `udemy_courses_eda.ipynb` — the full analysis notebook  
- `udemy_courses.csv` — dataset used in the notebook
